# vgtcentos
Provisionamento de uma máquina CENTOS7 com bash

Info
* Provider: virtual box
* Box: matyunin/centos7
* Provisioning: bash

Scripts disponíveis
* Apache (10080) + VHOSTs
* PHP
* Composer
* NPM
* Bower
* Symfony installer
* Apps
* + Lynx
* + Telnet
* MongoDB (27017)
